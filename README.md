# MoviePilot 自动签到插件

一个支持贴吧和微博超话自动签到的 MoviePilot V2 插件。

## 功能特性

- ✅ **贴吧自动签到** - 支持一键签到和逐个签到
- ✅ **微博超话自动签到** - 自动签到所有关注的超话
- ✅ **定时任务** - 每天定时自动执行签到
- ✅ **结果通知** - 签到结果通过 MoviePilot 通知系统推送
- ✅ **手动触发** - 支持 API 手动触发立即签到
- ✅ **连接测试** - 可测试 Cookie/BDUSS 是否有效

## 插件目录结构

```
plugins/v2/autosign/
├── __init__.py          # 插件主类
├── tieba_sign.py        # 贴吧签到模块
└── weibo_sign.py        # 微博超话签到模块
```

## 安装方法

### 方法一：插件市场安装（推荐）

1. Fork 本仓库到你的 GitHub
2. 在 MoviePilot 后台 -> 设置 -> 插件市场 -> 第三方插件市场，添加你的仓库地址
3. 在插件市场中找到"自动签到"插件，点击安装

### 方法二：手动安装

1. 将 `plugins/v2/autosign` 目录复制到 MoviePilot 的 `app/plugins` 目录下
2. 重启 MoviePilot 或开启插件热加载
3. 在插件列表中找到并启用插件

## 配置说明

### 基础设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 是否启用自动签到插件 | 是 |
| 每日签到时间 | 每天自动签到的时间（24小时制） | 08:00 |

### 贴吧签到设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用贴吧签到 | 是否启用贴吧签到 | 是 |
| BDUSS | 百度 BDUSS Cookie（必填） | - |
| STOKEN | 百度 STOKEN Cookie（可选） | - |
| 优先使用一键签到 | 优先使用贴吧一键签到功能（VIP功能） | 是 |
| 签到间隔 | 每个贴吧签到之间的延迟（秒） | 1.5 |

### 微博超话签到设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用微博超话签到 | 是否启用微博超话签到 | 否 |
| 微博 Cookie | 微博完整 Cookie（必填） | - |
| 签到间隔 | 每个超话签到之间的延迟（秒） | 2 |

### 通知设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用签到结果通知 | 是否发送签到结果通知 | 是 |
| 仅失败时通知 | 只有签到失败时才发送通知 | 否 |

## 获取 Cookie 方法

### 获取贴吧 BDUSS

1. 浏览器登录 [百度贴吧](https://tieba.baidu.com/)
2. 按 F12 打开开发者工具
3. 切换到 Network（网络）标签
4. 刷新页面，点击任意请求
5. 在 Request Headers 中找到 Cookie
6. 复制 `BDUSS=` 后面的值（到分号为止）

### 获取微博 Cookie

1. 浏览器登录 [微博](https://weibo.com/)
2. 按 F12 打开开发者工具
3. 切换到 Network（网络）标签
4. 刷新页面，点击任意 weibo.com 的请求
5. 在 Request Headers 中找到 Cookie
6. 复制完整的 Cookie 字符串

## API 接口

插件提供以下 API 接口：

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/plugin/autosign/sign_now` | POST | 立即执行签到 |
| `/api/v1/plugin/autosign/sign_result` | GET | 获取上次签到结果 |
| `/api/v1/plugin/autosign/check_tieba` | GET | 测试贴吧连接 |
| `/api/v1/plugin/autosign/check_weibo` | GET | 测试微博连接 |

## 注意事项

1. **Cookie 有效期**：微博和贴吧的 Cookie 都会过期，需要定期更新
2. **请求频率**：不要设置太短的签到间隔，避免被封号
3. **账号安全**：不要在公共环境使用，Cookie 等同于密码
4. **反爬机制**：网站可能随时更新接口，脚本需要跟进维护
5. **法律风险**：自动化签到可能违反平台用户协议，请谨慎使用
6. **贴吧一键签到**：0 点到 1 点不能使用一键签到，且只能签到 7 级以上贴吧

## 常见问题

### Q: 为什么签到失败？
A: 可能的原因：
- Cookie/BDUSS 已过期，需要重新获取
- 网络问题，检查服务器网络连接
- 账号被限制，等待一段时间再试
- 网站接口更新，需要更新插件版本

### Q: 可以多账号签到吗？
A: 当前版本暂不支持多账号，如需多账号可以部署多个 MoviePilot 实例或等待后续版本更新。

### Q: 签到失败会重试吗？
A: 当前版本不会自动重试，建议设置合理的签到时间，并开启失败通知，发现失败后手动触发重试。
